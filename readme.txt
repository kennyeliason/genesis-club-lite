Mobile Responsive Logos, Hamburger Menus, Animated Top Bars, FAQ Accordions, User Signatures, Google Calendars and much more for Genesis sites.

== Description ==

**IMPORTANT:** This plugin can ONLY be used with Genesis child themes.

**MAINTENANCE UPDATE:** This is a maintained fork of the original Genesis Club Lite plugin, updated for compatibility with the latest versions of WordPress and Genesis. The original repository has been archived, and this fork includes updated code, fixes for deprecated functions, and code formatting improvements.

**SUPPORT:** For continued support and updates, you can follow this maintained fork.

**FEATURES:**

Genesis Club Lite is organized into separate modules so you can just load the modules you need.

DASHBOARD MODULE:
* Switch on only the modules you need - there is no bloat in this plugin.
* Enable the plugin selectively for any custom post types you have defined.

== Changelog ==

= 1.17.1 =
* **Maintenance:** Updated code for compatibility with WordPress 6.3 and Genesis 3.x.
* **Fix:** Corrected formatting issues in multiple files for better readability and maintainability.
* **Compatibility:** Validated compatibility with PHP 8.0 and above.
* **Update:** Tested up to WordPress 6.3.
* **Deprecated Functions:** Updated functions to avoid deprecated warnings in the latest PHP versions.

= 1.17 =

* Added Google Tag Manager support for both public and membership sites.
* Added option for Clicky script addition for non-logged-in pages.
* Added option to remove version numbers from script and stylesheet SRC query strings.
* Added Facebook comments widget.
* Added color options for labels and login button on WordPress login.
* Added option for home page only script.
* Added instance level class to Genesis Club widgets.
* Added option to hide or show the top bar on taxonomy archive pages.
* Consolidated breadcrumbs, postinfo, postmeta, excerpt images, and sort order on to a single Display tab on archive pages.
* Updated simple social icons support to use SVG files rather than fonts.
* Validated PHP 7 compatibility.
* Updated for WordPress 4.8 and Genesis 2.5.2.
